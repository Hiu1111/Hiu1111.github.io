02 — Selected Work

Things I've built recently.

/ 01

Pursuit — AI Job Tracker

An AI-powered job application tracker that processes natural language commands and automates multi-step application workflows through an agent-based architecture. Backed by the Anthropic Claude API and synced to Google Sheets in real time for persistent tracking and analytics.

Python Flask Claude API Google Sheets API JavaScript HTML/CSS
/ 02

Detour — Smart Flight Routing

A mobile app that rethinks budget travel across Asia by surfacing multi-city stopover routes that often cost less than direct flights — while letting travelers explore more destinations along the way. Pulls real-time fares from the Amadeus API and ranks itineraries by cost, layover convenience, and total travel time.

Mobile Amadeus API Route Optimization Real-time Data
/ 03

Relay — Cloud ETL Pipeline

An automated ETL pipeline that ingests, transforms, and loads sales data into PostgreSQL on Amazon RDS using AWS Glue and Python — orchestrated end-to-end with Jenkins CI/CD for consistent, repeatable runs. Cloud infrastructure (RDS, S3, IAM) is provisioned with Terraform and CloudFormation, and the pipeline auto-generates Excel reports via OpenPyXL to make the transformed data easy to analyze.

AWS Glue AWS Lambda RDS / S3 Terraform CloudFormation Jenkins CI/CD Python SQL
/ 04

Sentinel — Cloud QA Platform

An end-to-end automated testing platform that catches regressions before they ship. API smoke tests run with RestAssured, UI flows are covered by Selenium WebDriver, and a Jenkins Pipeline-as-Code setup auto-triggers the full suite on every code change. JUnit reports and archived test artifacts make debugging fast and give the team real visibility into release health.

Java Selenium RestAssured JUnit Jenkins Maven Docker S3
/ 05

Lens — Live Emotion Recognition

A computer vision system that reads emotions off a live webcam feed in real time. Built with Python and OpenCV for video capture and processing, with DeepFace classifying expressions into 3+ categories (happy, sad, neutral, and more). Integrated MTCNN for accurate face localization, which dramatically improves detection reliability in low-light and noisy conditions.

Python OpenCV DeepFace MTCNN Computer Vision Real-time
